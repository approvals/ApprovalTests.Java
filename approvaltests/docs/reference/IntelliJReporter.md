## Explanation

Finding IntelliJ is a complicated thing because the path is always changing.
This page exists when IntelliJ is not being found in your system.


## Current Support

The IntelliJReporter currently will find IntelliJ installations if and only if it is currently running, this functionality requires Java 9 or higher.
